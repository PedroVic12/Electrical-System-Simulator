## PandaPower - Template - Dashbord Simulador de Redes Elétricas
---

✨ Características Principais:
🎨 Design Moderno:

Interface com gradiente e efeitos de blur (glassmorphism)
Cards estatísticos com animações hover
Layout responsivo para diferentes dispositivos
Cores e tipografia profissionais

📊 Visualizações Interativas:

Diagrama Unifilar: Mostra a topologia do sistema com barras, linhas e transformadores
Gráfico de Potências: Potências ativa e reativa de linhas e transformadores
Perfil de Tensões: Tensões das barras com limites operacionais (linhas vermelhas)
Correntes Elétricas: Correntes das linhas e transformadores em gráfico de barras

📈 Recursos Técnicos:

Cards de Status: 14 barras, 15 linhas, 5 transformadores e status do sistema
Indicador de Status: Verde (normal), amarelo (atenção), vermelho (crítico)
Dados Exatos: Todos os valores extraídos diretamente do seu arquivo Plotly
Interatividade: Hover, zoom, pan nos gráficos
Responsividade: Adapta-se a tablets e smartphones

🔧 Funcionalidades:

Monitoramento automático dos limites de tensão (0.95-1.05 pu)
Atualização periódica (simulada a cada 30 segundos)
Redimensionamento automático dos gráficos
Interface intuitiva para engenheiros elétricos

O dashboard está pronto para uso em sistemas reais de monitoramento, bastando conectar com APIs ou bancos de dados para atualizações em tempo real dos parâmetros elétricos!
